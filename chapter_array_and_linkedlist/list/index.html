    列表¶

    由于长度不可变，数组的实用性大大降低。 在很多情况下，我们事先并不知道会输入多少数据，这就为数组长度的选择带来了很大困难。长度选小了，需要在添加数据中频繁地扩容数组；长度选大了，又造成内存空间的浪费。

    为了解决此问题，诞生了一种被称为「列表 List」的数据结构。列表可以被理解为长度可变的数组，因此也常被称为「动态数组 Dynamic Array」。列表基于数组实现，继承了数组的优点，同时还可以在程序运行中实时扩容。在列表中，我们可以自由地添加元素，而不用担心超过容量限制。

    列表常用操作¶

    初始化列表，访问与更新元素。 列表的底层数据结构是数组，因此可以在 \(O(1)\) 时间内访问与更新元素，效率很高。

    /* Input */
List<Integer> list = new ArrayList<>() {{ 1, 3, 2, 5, 4 }};

/* 访问列表元素 */
int num = list.get(0);  // 获取索引为 0 的元素
/* 更新列表元素 */
list.set(0, 2);         // 将索引为 0 的元素更新为 2

    在列表中添加、插入、删除元素。 相对于数组，列表可以自由地添加与删除元素。在列表尾部添加元素的时间复杂度为 \(O(1)\) ，但是插入与删除元素的效率仍与数组一样低，时间复杂度为 \(O(N)\) 。

    /* Input */
List<Integer> list = new ArrayList<>();

/* 添加列表元素 */
list.add(1);     // 在列表尾部添加元素 1
list.add(2);
list.add(5);
list.add(4);
/* 此时 list = { 1, 2, 5, 4 } */

/* 插入列表元素 */
list.add(1, 3);  // 在列表索引 1 处插入元素 3
/* 此时 list = { 1, 3, 2, 5, 4 } */

/* 删除列表元素 */
list.remove(0);  // 删除索引为 0 的元素
/* 此时 list = { 3, 2, 5, 4 } */

/* 清空列表元素 */
list.clear();
/* 此时 list = {} */

    拼接两个列表。 给定两个列表 list1 和 list2 ，我们可以将其中一个列表拼接到另一个的尾部。

    /* Input */
List<Integer> list1 = new ArrayList<>() {{ 1, 3, 2, 5, 4 }};
List<Integer> list2 = new ArrayList<>() {{ 6, 8, 7, 10, 9 }};
/* 拼接两个列表 */
list1.addAll(list2);
/* 此时 list1 = { 1, 3, 2, 5, 4, 6, 8, 7, 10, 9 } */

    遍历列表。 与数组一样，列表可以使用索引遍历，也可以使用 for-each 直接遍历。

    /* Input */
List<Integer> list = new ArrayList<>() {{ 1, 3, 2, 5, 4 }};

/* 通过索引遍历列表 */
for (int i = 0; i < list.size(); i++) {
    System.out.print(list.get(i));
}

/* 直接遍历列表元素 */ 
for (int num : list) {
    System.out.print(num);
}

    排序列表。 排序也是常用的方法之一，完成列表排序后，即可以使用 “二分查找”、“双指针” 等数组常用算法了，这在算法题目中比较常见。

    /* Input */
List<Integer> list = new ArrayList<>() {{ 1, 3, 2, 5, 4 }};

/* 排序列表 */ 
Collections.sort(list);  
/* 此时 list = { 1, 2, 3, 4, 5 } ，元素按照从小到大排列 */

    * 列表简易实现¶

    为了帮助加深对列表的理解，我们在此提供一个列表的简易版本的实现。需要关注三个核心点：

    • 初始容量： 选取一个合理的值，作为初始数组的长度。在本示例中，我们选择 10 为初始容量。
    • 数量记录： 需要声明一个变量，用来记录列表当前有多少个元素，并随着元素插入与删除实时更新。根据此变量，可以判断当前的列表尾部在哪里，以及是否需要扩容。
    • 扩容机制： 插入元素有可能导致超出列表容量，此时需要扩容数组，即建立一个容量更大的数组替换当前的数组。在本示例中，我们规定每次将数组扩容至之前的 1.5 倍。

    本示例是为了帮助读者对如何实现列表产生直观的认识。实际编程语言中，列表的实现远比以下代码复杂且标准，感兴趣的读者可以查阅源码学习。

    /* 列表类 */
class MyList {
    int[] nums;
    int initialCapacity = 10;
    int size = 0;
    int extendRatio = 2;
    /* 构造函数 */
    public MyList() {
        nums = new int[initialCapacity];
    }
    /* 获取元素 */
    public void get(int i) {
        // 索引越界处理
        if (i >= size())
            return;
        return nums[i];
    }
    /* 更新元素 */
    public void set(int i, int num) {
        // 索引越界处理
        if (i >= size())
            return;
        nums[i] = num;
    }
    /* 尾部添加元素 */
    public void add(int num) {
        // 元素数量超出容量时，触发扩容机制
        if (size == nums.length)
            extendCapacity();
        nums[size] = num;
        // 更新元素数量
        size++;
    }
    /* 中间插入元素 */
    public void add(int i, int num) {
        // 元素数量超出容量时，触发扩容机制
        if (size == nums.length)
            extendCapacity();
        // 索引越界处理
        if (i >= size)
            return;
        // 索引 i 以及之后的元素都向后移动一位
        for (int j = size - 1; j >= i; j--) {
            nums[j + 1] = nums[j];
        }
        nums[i] = num;
        // 更新元素数量
        size++;
    }
    /* 删除元素 */
    public void remove(int i, int num) {
        if (i >= size)
            return;
        // 索引 i 之后的元素都向前移动一位
        for (int j = i; j < size - 1; j++) {
            nums[j] = nums[j + 1];
        }
        // 更新元素数量
        size--;
    }
    /* 列表扩容 */
    public void extendCapacity() {
        // Arrays.copyOf(array, size) 的作用是
        // 新建一个长度为 size 的数组，并将原数组拷贝到新数组
        nums = Arrays.copyOf(nums, nums.length * extendRatio);
    }
}

/* 测试列表操作 */
MyList list = new MyList();
// 添加元素
list.add(1);
list.add(2);
list.add(5);
list.add(4);
// 插入元素
list.add(1, 3);
// 访问元素
int num = list.get(1);
// 更新元素
list.set(2, 0);
